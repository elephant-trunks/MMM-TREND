# MMM-TREND
MMM-TREND is a multilingual, multimodal misinformation dataset designed for temporal and spatial analysis of misinformation narratives. The dataset supports research in multimodal detection, cross-lingual modelling, and longitudinal topic analysis.

# Image Data

Due to repository size constraints, image files are not hosted in this repository.

Images corresponding to claim identifiers are available for anonymous review at:

# Ethical Note

The dataset contains misinformation content collected strictly for academic research purposes. Users are advised to handle the data responsibly. It is protected under CCL.

# Anonymity Notice

This repository is prepared for double-blind review. All identifying information has been removed.
